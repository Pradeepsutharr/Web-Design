1. form tag in HTML is used to create an interactive form on a webpage.
  2. The form tag in HTML is used to collect data from the user and send it to the server for further processing.
  3. It is used with other elements such as input, textarea, and submit, which are used to make a complete form.

  4. name="" attr will help to send form element data to the server
  5. input type="submit" will post form data to the server
  6. there are two methods of send user user data to the server in FORM tag

GET method

  • Appends form-data into the URL in name/value pairs
  • The length of a URL is limited (about 3000 characters)
  • Useful for form submissions where a user wants to bookmark the result
  • GET will use URL for sending client data to the server its less secure
  • GET is better for non-secure data, like query strings in Google
  • Not supports file upload
  • mainly we use it for send one page data to other and also use it for searching

Post Method

  • Appends form-data inside the body of the HTTP request (data is not shown in URL)
  • POST will send form data to the server without URL thats why its secure
  • it has no size limitations
  • support file upload
  • Form submissions with POST cannot be bookmarked
