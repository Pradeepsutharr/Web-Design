1. Tags that dont have a CLOSING TAG are called Self Closing or Void Elements
  2. Void elements cannot have content inside it.
  3. Void elements have attributes.
  4. Void elements cannot be nested.

The following are some of the void elements.

  • <img>

  • <input>

  • <link>

  • <meta>
  • <br>

  • <hr>

  • <area>

  • <embed>

  • <base>

  • <col>

  • <keygen> Deprecated

  • <param> Deprecated

  • <source>

  • <track>

  • <wbr>
