Child selector

  1. The child selector selects all elements that are the children of a specified element.
  2. The child combinator (>) is placed between two CSS selectors. It matches only those elements matched by the second selector that are the direct children of elements matched by the first.
  3. in the following example < a> tag is child of the < div>
    < div class="custom-class"> Lorem ipsum dolor, sit amet consectetur < a href="#">Site Title< /a> Magni ab nam nesciunt maiores? < /div>
