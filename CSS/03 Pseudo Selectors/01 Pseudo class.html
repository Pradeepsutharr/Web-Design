Pseudo class selector

  • A Pseudo class in CSS is used to define the special state of an element.

  • It can be combined with a CSS selector to add an effect to existing elements based on their states.

  • For Example, changing the style of an element when the user hovers over it, or when a link is visited. All of these can be done using Pseudo Classes in CSS.

Pseudo classes

  1. :hover Pseudo-class
    This pseudo-class is used to add a special effect to an element when our mouse pointer is over it.

  2. :active Pseudo-class
    This pseudo-class is used to select an element that is activated when the user clicks on it.

  3. :focus Pseudo-class
    This pseudo-class is used to select an element that is currently focused by the user.

  4. :visited Pseudo-class
    This pseudo-class is used to select the links which have been already visited by the user. the color of the link changes once it is visited.

  5. :not pseudo-class
    This pseudo-class is used to select elements that do not match a specific selector.
